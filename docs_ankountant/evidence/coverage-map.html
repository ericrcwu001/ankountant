Abstain Rule

The readiness backend abstains below the coverage line and below the sealed-attempt volume line. The current rule is: no readiness score with fewer than 20 sealed attempts or less than 60% topic coverage. Backend tests assert that abstained responses clear score fields while keeping the reason and give-up evidence.
